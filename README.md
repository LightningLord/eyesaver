# Eyesaver

Simple reminder gem to save your eyes. Based on the 20-20-20 guideline.

## Installation

`$ gem install eyesaver`

## Usage

`Eyesaver.run!`

You will be reminded every 20 minutes to rest your eyes by a browser window.
